# AI 예측 마스크로 자동 측정 (테스트셋 검증용)

## 무엇이 되나
1. **예측 마스크 폴더 연결** — pred_masks/test 폴더를 그대로 연결.
   파일명 규칙 자동 인식: `{원본}_pred_mask.png` (그 외 `_pred`, `_mask`, `_predmask`도 인식)
2. **AI로 측정** — 그 이미지의 예측 마스크를 읽어
   - binary 마스크를 **연결요소로 추체 분리**
   - 각 덩어리의 외곽 윤곽선을 폴리곤으로 변환
   - 위→아래 순으로 라벨(시작 라벨 기준, 기본 C2) 부여
   - 기존 자동 측정 알고리즘(v1/v2 + 검증)에 그대로 투입
   → LL/TK/CL·분절각·쐐기각과 품질(정상/확인필요/축보정)이 동일하게 산출됩니다.
3. **마스크 겹쳐보기** — 추출된 폴리곤을 청록색으로 오버레이해 분리가 제대로 됐는지 확인.
4. **테스트셋 안내** — 현재 이미지의 예측 마스크가 폴더에 없으면
   "이 이미지는 예측 마스크 없음 — 테스트셋이 아닐 수 있습니다"라고 표시합니다.
   (학습/검증셋 이미지에서 잘못 돌리는 것을 방지)

## 사용 순서
1. 왼쪽 패널 "AI 예측 마스크" → [폴더 연결] → pred_masks/test 선택
2. 이미지를 열면 마스크 유무가 자동 표시됨
3. [AI로 측정] 클릭 → 결과표/오버레이가 AI 기준으로 갱신
4. 사람 라벨 기준으로 되돌리려면 [자동 측정 실행]

## 사람 라벨과 비교하는 법
- [자동 측정 실행] → 결과 확인 → [AI로 측정] → 같은 지표 비교
- 각각 [CSV]로 내보내 정량 비교 가능

## 검증 (합성 데이터)
- 추체 5개 + 노이즈 마스크 → 노이즈 제외하고 5개 정확 분리, 폴리곤 좌표 일치
- 그 폴리곤을 자동 측정에 투입 → v1/v2 모두 정상 산출(품질 전부 ok)

## 주의
- binary 마스크는 추체 구분이 없어 **붙어 있는 추체는 한 덩어리**로 잡힙니다.
  이 경우 분리 개수가 실제와 달라지므로 "마스크 겹쳐보기"로 꼭 확인하세요.
- 라벨은 위에서부터 순서대로 부여되므로, 시작 라벨(왼쪽 '시작 척추뼈')이 정확해야 합니다.
